# **App Name**: DalyReview

## Core Features:

- Admin Authentication: Secure admin panel access using provided credentials (bpskar2@gmail.com, 12@techxera).
- Daily Update: Teachers update daily teaching plans via the admin panel.
- Student View: Students can view the daily teaching updates for each subject.
- Student Login: Students login and sign-up with name, registration number, year, section, password, and 'show password' option with 'forgot password' option.
- Subject Review: Students can submit reviews for each subject.
- Admin Review Dashboard: Admin panel displays student reviews for each subject.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to provide a sense of trust and reliability for the educational context.
- Background color: Light blue (#E3F2FD) provides a soft, non-distracting backdrop.
- Accent color: Teal (#4DB6AC) for important interactive elements.
- Font pairing: 'Inter' (sans-serif) for body text and 'PT Sans' (sans-serif) for headlines, ensuring readability and a modern feel.
- Use simple, clear icons to represent subjects and review actions.
- Clean and intuitive layout to ensure ease of navigation for both teachers and students.
- Subtle transitions and feedback animations on user interactions to enhance the user experience.
